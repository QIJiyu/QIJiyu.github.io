最近一个项目是做金融的, 需要实时更新数据, 在 ChartData 里面发现了

官方注释说可以根据 DataSet 的索引删除某个条目的对象, 那么我们开始

需求是对某支股票的点进行实时刷新, 也就是如果最后的一个点有更新, 我只需要把它从 ChartData 删掉, 再加上更新的数据, 刷新 ChartView 就可以了

下面是分时图里部分代码, 由于分时图存在分时线和均线, 需要分别取出, 分别操作

除此之外, ChartData 里面还提供了

• 新增一个横坐标
- (void)addXValue:(NSString * _Nullable)xVal;

• 删除一个横坐标
- (void)removeXValue:(NSInteger)index;

• 新增一组数据
- (void)addDataSet:(id_Null_unspecified)d;

• 删除一组数据
- (BOOL)removeDataSet:(id_Null_unspecified)dataSet;

• 根据索引删除一组数据
- (BOOL)removeDataSetByIndex:(NSInteger)index;

• 根据 DataSet 索引和点在 DataSet 下的索引删除一个点
- (BOOL)removeEntryByXIndex:(NSInteger)xIndex dataSetIndex:(NSInteger)dataSetIndex;

这些用法都大同小异, 只要清楚了解你要操作的数据, 使用起来还是很方便的
